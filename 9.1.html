Қысқаша теориялық материалдар

Ішкі программаларды құру және оларды қолдану

Практикалық есептерді шығару барысында үлкен программаның белгілі бір бөлігін әртүрлі мәндер бойынша бірнеше қайталап пайдалануға тура келеді. Жеке программа түрінде бөлек жазып, қажет кезінде оны шақырып, пайдалануға болатын негізгі программаның арнайы жазылған бөлігін қосалқы программа (подпрограмма) деп атайды.

Функция белгілі бір тапсырманы орындайтын және бағдарламаның басқа бөліктерінде қайталап қолдануға болатын код блогы болып табылады.

 

Python программалау тіліндегі функцияны қарастырайық.

Функцияның жазылу форматы:

def функция_аты ([параметрлер]):

    инструкциялар

 

Жақша ішіндегі параметрлер міндетті емес.

Қарапайым функцияны қарастырайық:

def say_hello():


    print("Hello")

 

Функцияны шақыру үшін функция аты және жақшаның ішінде параметрлерге берілетін мәндер көрсетіледі.

def say_hello():


    print("Hello")

     


say_hello()


say_hello()


say_hello()

 

Мұнда say_hello функциясы үш рет шақырылып тұр.

 

Енді функцияны параметрлерімен қолданайық.

def say_hello(name):


    print("Hello,",name)


 

say_hello("Tom")


say_hello("Bob")


say_hello("Alice")


 

Үнсіз келісім бойынша мәндер

Функцияның кейбір параметрлерінің мәндерін үнсіз келесім бойынша қабылдайтындай міндетті емес етуге болады. Мысалы:

def say_hello(name="Tom"):


    print("Hello,", name)


     

say_hello()


say_hello("Bob")

 

Мұнда name параметрі міндетті емес болып табылады. Егер функцияны шақырған кезде оған мән бермейтін болсақ, онда үнсіз келісім бойынша Tom жолы шығады.

main функциясы

Бағдарламада көптеген функциялар болуы мүмкін. Сондықтан оларды реттеу үшін арнайы main функциясын қосып, оның ішіне басқа функцияларды жазған дұрыс.

def main():


    say_hello("Tom")


    usd_rate = 56


    money = 30000


    result = exchange(usd_rate, money)


    print("Берілетін ақша", result, "доллар")

 

 

def say_hello(name):


    print("Hello,", name)

     

     

def exchange(usd_rate, money):


    result = round(money/usd_rate, 2)


    return result

 

# main функциясын шақыру  


main()

 

Енді С++ программалау тіліндегі функцияны қарастырайық.

Си тілінде  қосалқы программаны жазу функция арқылы жүзеге асырылады. Си тілінде функцияны екі жолмен сипаттауға болады: main функция­сына дейін (бұл қысқаша сипатталуы) және main функциясынан кейін немесе  оған дейін (анықтау ретінде). Функция - программаның қысқаша түрдегі жазылу форматы төмендегідей:

 

<нәтиженің типі> <функцияның  аты>(<тип> <1-айнымалы>,

<тип> <2-айнымалы>, …,<тип> <N-айнымалы>);

 

Бұл жағдайда дөңгелек жақша ішіндегі айнымалылардың атауларын көрсетпей, тек олардың типтерін ғана көрсетуге болады, өйткені олар компилятормен өңделмейді.

main сөзінен кейін жазылған функция программаның толық жазылу форматы төмендегідей:

 

<нәтиженің типі> < функцияның аты >(параметрлер тізімі)

{

функция денесі

}

 

Мұндағы нәтиженің типі фунцияның қабылдайтын мәнінің типін көрсетеді. Басқа программалау тілдеріндегідей Си тілінде де функция программадан алынатын нәтиже біреу.  Ол нәтиже return (ағылшынның оралу, возвращать сөзінен алынған) операторы арқылы алынады. Функция-программа функция атауы арқылы шақырылады. Дөңгелек жақша ішіндегі параметрлер тізімі үтір арқылы ажыратылып жазылады. Кей жағдайда функция параметрсіз жазылуы мүмкін. Мұндай жағдайда параметрлер болмаса да дөңгелек жақша міндетті түрде жазылуы тиіс. Мұндағы return операторы функцияны шақырып, қажетті мәнді қабылдағаннан кейін программадағы басқаруды функцияны шақырудан кейінгі тұрған операторға береді. Кей жағдайларда return операторы жазылмайды. Егер функция программадан алынатын мәннің типі көрсетілмеген жағдайда, ол бүтін (int) типті болып есептеледі.
