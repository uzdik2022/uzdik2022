Теориялық материалар.

Кіріктірілген шарттарды программалау үшін бірнеше тәсілдерді қолдануға болады. Оны бірнеше шартты операторды қолдану арқылы жүзеге асыра аламыз. Кей жағдайларда шартты оператордың ішінде шартты оператор орналасуы мүмкін. if операторы басқа if операторының ішінде болса, онда оны кіріктірілген шарт деп атайды. Мысалы,

Python программалау тіліндегі жазылу форматы:

if <1 шартты өрнек> :

    <1 оператор> 

elif <2 шартты өрнек> :

    <2 оператор>

else:

    <3 оператор>

 

C++ программалау тілінде шартты көшу оператордың жазылу түрі:

if <1 шартты өрнек>  <1 оператор>; 

    else if  <2 шартты өрнек>    <2 оператор>; 

                 else <3оператор>; 

 

Жай шарт – =, <>, <=, >=, <, > қатынас таңбаларының бірімен біріктірілген екі өрнек (логикалық өрнек).

Құрама шарт – ЖӘНЕ, НЕМЕСЕ, ЕМЕС қызметші сөздері арқылы біріктірілген екі немесе одан да көп жай шарттар.

Мысалы: (x>0) және (у>4) және (x+y≤4) (x<2) немесе (x>5) емес(x>3) және (X≤6)

AND - ЖӘНЕ

OR – НЕМЕСЕ

NOT – ЕМЕС

Қойылған шарттың тексерілу нәтижесі ақиқат болатын болса, онда өрнек нәтижесінің мәні 1-ге,  кері жағдайда 0-ге тең болады.

Логикалық операциялар. ІF сөзінен кейін жазылатын шарт екі немесе одан да көп болатын болса, онда логикалық қатынас белгілері (операциялар) пайдаланылады. Программалау тілдерінде пайдаланылып жүрген логикалық қатынас белгілерінің С/С++ тілдеріндегі жазылу түрі төмендегідей:
